# kitconcst
* streamlit==1.57.0
* kitconc==3.4.4
  
O kitconcst foi construído com base no  um pacote de ferramentas voltado para Linguística de Corpus e análise de texto, utilizado para criar linhas de concordância e analisar corpora, e a biblioteca **Streamlit**.
Para realizar a criação de um corpus manipulável pela ferramenta, é preciso deﬁnir uma pasta de trabalho (workspace), um nome de identiﬁcação para o corpus e o idioma dos textos. É preciso também adicionar textos ao corpus criado por meio de uma pasta.

Nas atividades desenvolvidas aqui, não se faz necessário criar nenhuma dessas pastas, pois elas são criadas automaticamente quando executamos esta interface. Confira na barra ao lado clicando "no símbolo de uma pasta" e você verá que há duas pastas Corpus, onde colocaremos os arquivos do nosso corpus, e a workspace.
